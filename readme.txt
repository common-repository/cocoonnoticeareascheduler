テーマ「Cocoon」で動作するプラグインです。Cocoonの通知エリアの設定を複数設定できます。

== Description ==


このプラグインでは通知メッセージを複数登録することができ、表示期間・投稿記事ID・カテゴリーID指定で表示の制御ができます。

背景色・文字色・リンク先・新規タブに開くかなどもCocconの機能と同様に設定できます。

スケジュール機能はサーバー環境、プラグインなどのキャッシュ機構によっては時間通りに切り替わらない可能性があります。

Cocoon以外では動作しませんが、編集などは可能です。

Cocoonのカスタマイズ機能を利用しているため、テーマ側の仕様変更により利用できなくなる可能性があるのでご了承ください。

* [詳細な説明はこちら](https://dev.macha795.com/wp-plugin-notice-manage/)


== Installation ==

1. WordPressの管理画面から「プラグイン」→「新規追加」をクリックします。
2. ブラウザの入力ボックスに「CocoonNoticeAreaScheduler」と入力します。
3. CocoonNoticeAreaScheduler」プラグインを選択し、「インストール」をクリックします。
4. プラグインを有効にします。

または、

1. このページからプラグインをダウンロードします。
2. .zipファイルをパソコンのある場所に保存します。
3. WordPressの管理画面を開き、「プラグイン」→「新規追加」をクリックします。
4. 「アップロード」をクリックして、このページからダウンロードした.zipファイルを参照します。
5. 「インストール」をクリックし、「プラグインを有効化」をクリックします。
